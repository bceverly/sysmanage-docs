Navigation & User Experience

The interface is designed for ease of use with intuitive navigation, responsive design, and comprehensive search capabilities.

  • Responsive Design: Works seamlessly on desktop, tablet, and mobile devices
  • Real-time Updates: Live data updates via WebSocket connections
  • Global Search: Quickly find hosts, packages, or users across your infrastructure
  • Multi-language Support: Interface available in 14 languages
  • Dark/Light Themes: Choose your preferred interface theme

Getting Started with the Web Interface

  1. Login: Access the web interface at https://your-server:8443
  2. Dashboard: Review the main dashboard for an overview of your infrastructure
  3. Approve Agents: Navigate to Hosts to approve pending agent connections
  4. Explore Features: Use the navigation menu to explore different management areas
  5. Customize: Adjust settings, language preferences, and notification options
