Health Analysis Module

AI-powered system health scoring with actionable recommendations to optimize your infrastructure performance and reliability.

Overview

The Health Analysis module uses advanced AI algorithms to analyze your systems' configuration, performance metrics, and security posture. It produces a comprehensive health score and provides prioritized recommendations for improvement.

Each host receives a health score from 0-100, calculated by analyzing multiple factors:

70-79 C Fair - Attention needed
60-69 D Poor - Immediate action recommended
0-59 F Critical - Urgent intervention required

Issue Detection

The health engine automatically detects and categorizes issues by severity:

Critical Immediate attention required - security risks or system failures
Warning Should be addressed soon - potential problems identified
Info Optimization opportunities - best practice recommendations

Actionable Recommendations

Each health analysis includes prioritized recommendations:

  • Priority levels (High, Medium, Low)
  • Categorized by type (Security, Performance, Configuration)
  • Specific actions to take
  • Expected impact on health score

Using Health Analysis

Viewing Health Scores

Health scores are displayed in two locations:

  • Hosts List - Health grade column shows at-a-glance status
  • Host Detail - Health tab provides full analysis details

Refreshing Analysis

Health analysis can be refreshed on-demand by clicking the "Run Analysis" button on the Health tab. Analysis results are cached and reused until a new analysis is requested.

Health History

Track health score trends over time to monitor improvement or identify declining system health. Historical data helps identify patterns and validate remediation efforts.

API Access

Health analysis is available via the REST API:

# Get health analysis for a host
GET /api/host/{host_id}/health-analysis

# Run new analysis
POST /api/host/{host_id}/health-analysis

# Get analysis history
GET /api/host/{host_id}/health-analysis/history
