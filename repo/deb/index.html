SysManage APT Repositories

Debian/Ubuntu package repositories for SysManage components.

SysManage Agent Available

Description: Agent that runs on managed systems

Repository: agent/deb/

Documentation: README

Installation:

echo "deb [trusted=yes] https://bceverly.github.io/sysmanage-docs/repo/agent/deb/apt stable main" | sudo tee /etc/apt/sources.list.d/sysmanage-agent.list
sudo apt update
sudo apt install sysmanage-agent
